# URL Shortener

## Backend setup

### Creating the env for Flask
``` properties

<!-- Run once only -->
python -m venv env
./env/Scripts/activate # For Windows
python -m pip install --upgrade pip
pip install -r requirements.txt

<!-- To start flask -->
source env/bin/activate # OR
./env/Scripts/activate # For Windows

flask run
```

Helpful commands
``` properties
pip install flask smth...
pip freeze > requirements.txt
```
