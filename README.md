# URL Shortener

## Backend setup

### Creating the env for Flask
``` properties

<!-- Run once only -->
python -m pip install --upgrade pip
python -m venv env
pip install -r requirements.txt

<!-- To start flask -->
source env/bin/activate # OR
./env/Scripts/activate # For Windows

flask run
```

## Frontend
``` properties
ng serve // Goto http://localhost:4200/

```
